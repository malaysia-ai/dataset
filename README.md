## Collection of dataset used by Malaysia AI

- This repo is for curating dataset related to Malaysia from multiple domain such as Tabular, Image, Text and Audio.

## Text

1. [huseinzol05/malay-dataset](https://github.com/huseinzol05/malay-dataset), gather Bahasa Malaysia corpus.
2. [huseinzol05/malaya-graph/data](https://github.com/huseinzol05/malaya-graph/tree/master/data), gather Bahasa Malaysia knowledge graph.

## Audio

1. [GoogleTTS](audio/googleTTS/)
2. [huseinzol05/malaya-speech/data](https://github.com/huseinzol05/malaya-speech/tree/master/data), gather Bahasa Malaysia speech.
