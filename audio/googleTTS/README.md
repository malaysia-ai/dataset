## Speech Synthesis from Google Translate

- This is for collecting bot speech data which might be use by chatbot company or telecommunication service

### SETUP

1. pip install git+https://github.com/khursani8/gTTS/@malay_lang // will use 'pip install gTTS' after they merge my PR
2. pip install smart_open
